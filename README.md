promypy
=======

A Promise A+ compatible library with cancellation and notification for python 2
or 3 with pyjsruntime

File: `promy.py`

Version: 1.0.0 - [License: WTFPLv2](#license)

Provides:

- [`Promise`](#promise)
- [`CancelException`](#cancelexception) (*non standard*)
